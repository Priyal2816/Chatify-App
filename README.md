# Chatify-App – Real-Time Full Stack Chat Application

🔗 **Live Demo:** https://chatify-app-eta.vercel.app  
📂 **GitHub:** https://github.com/Priyal2816/Chatify-App

---

## 📌 About
A full-stack real-time chat application built with the MERN stack and Socket.io — enabling instant one-on-one messaging with online/offline status, profile management, and secure JWT-based authentication.

---

## 🚀 Features

### 💬 Real-Time Messaging
- Instant one-on-one messaging using Socket.io
- Real-time message delivery with no page refresh
- Keyboard sound effects for immersive chat experience

### 👤 User Authentication
- Secure Signup & Login with JWT authentication
- Password hashing with bcrypt
- Protected routes for authenticated users only

### 🟢 Online Status
- See who is online/offline in real-time
- Active user list updates instantly

### 🖼️ Profile Management
- Upload and update profile pictures via Cloudinary
- View other users' profiles

### 📧 Email Integration
- Email notifications using Resend API
- Welcome emails on signup

### 🛡️ Security
- Arcjet security middleware for rate limiting & bot protection
- JWT-based session management

## 🚀 Getting Started

```bash
git clone https://github.com/Priyal2816/Chatify-App.git

# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm run dev
```
